# Break & Brews Frontend Admin UI

This directory contains the React TypeScript Vite application for the administrative dashboard.

## 🚀 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Dev Server**:
   ```bash
   npm run dev
   ```
   Open **[http://localhost:5173](http://localhost:5173)** in your browser.

3. **Production Build**:
   To compile and package the app for production deployment:
   ```bash
   npm run build
   ```

Refer to the main [Root README](../README.md) for database seeding and API server setup instructions.
